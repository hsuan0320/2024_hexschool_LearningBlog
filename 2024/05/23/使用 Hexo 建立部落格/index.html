使用 Hexo 建立部落格

在開始這段學習歷程之前，我一直在尋找一個簡單、高效的工具來記錄和分享我的前端學習筆記。經過一番研究，我決定使用 Hexo 來建立我的個人部落格。這篇文章將記錄我學習和使用 Hexo 的過程，希望能對其他學習者有所幫助。

為什麼選擇 Hexo

Hexo 是一個基於 Node.js 的靜態網站生成器，它有以下幾個優點：

  • 速度快：Hexo 可以在短時間內生成靜態文件。
  • 易於使用：Hexo 的指令簡單明瞭，容易上手。
  • 可擴展性強：可以通過主題和插件自定義網站的外觀和功能。

安裝 Hexo

首先，我需要安裝 Node.js 和 npm。可以從 Node.js 官網下載安裝包，並按照指示完成安裝。

接著，在終端中運行以下命令來安裝 Hexo：

npm install -g hexo-cli
