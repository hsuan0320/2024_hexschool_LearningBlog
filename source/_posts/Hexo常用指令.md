## Hexo 常用指令一覽

| 指令 | 描述 | 
| -------- | -------- |
| `hexo init <你的網站名稱>` | 初始化一個新的 Hexo 部落格 |
| `hexo server` | 啟動本地伺服器，預覽網站 |
| `hexo new "<部落格標題>"` | 新增一篇部落格文章 |
| `hexo generate` | 生成靜態網頁文件 |
| `hexo clean` | 清除快取和暫存文件 |
| `hexo deploy` | 部署網站至遠端伺服器 |
